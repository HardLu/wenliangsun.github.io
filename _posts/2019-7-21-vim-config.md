# Windows 篇

### 1. 安装 `Vim`

从网络[https://www.vim.org/download.php](https://www.vim.org/download.php)下载Vim的安装包。一路默认进行安装。安装完成之后，将vim的安装路径添加到系统的环境变量中，方便直接在CMD下进入Vim编辑器。安装完成之后在CMD下输入vim即可进入vim编辑器。如出现下图所示结果，说明安装成功。接下来配置vim编辑器。
