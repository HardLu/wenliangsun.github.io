## 最大公约数(欧几里得算法)

求两个正数的最大公约数，时间复杂度$O(logn)$.

### C++ 版本代码

```c++
// 最大公约数 a>b
int gcd (int a, int b){
    return b ? gcd(b, a % b) : a;
}

// 两个数的最小公倍数 * 两个数的最大公约数 = a*b a>b

// 最小公倍数
int lcm(int a, int b) {
    return a * b / gcd(a, b);
}
```
